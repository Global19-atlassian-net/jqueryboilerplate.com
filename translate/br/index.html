jQuery Boilerplate

Jump-starting jQuery plugins development

Contém muitos comentários para facilitar sua vida. É realmente orientado a objeto, implementa métodos e propriedades tanto públicos como privados, se tornando o candidato ideal para criação de plugins simples ou complexos para jQuery.

Não adere às sugestões feitas pela documentação do jQuery em relação ao Plugins/Authoring e fazendo isso provê melhor performance e uso de memória por não criar múltiplas instâncias de si mesmo e atribuindo elas ao DOM.

Modo de uso


$(document).ready(function() {

   // cria uma nova instância do plugin
   var myplugin = new $.pluginName($('#element'));

   // chama um método público
   myplugin.fooPublicMethod();

   // pega o valor de uma propriedade pública
   myplugin.settings.property;

});

jQuery Boilerplate (versão curta)

Você pode querer ver a versão totalmente comentada para um melhor entendimento.


;(function($) {

   $.pluginName = function(el, options) {

      var defaults = {
         propertyName: 'value',
         onSomeEvent: function() {}
      }

      var plugin = this;

      plugin.settings = {}

      var init = function() {
         plugin.settings = $.extend({}, defaults, options);
         plugin.el = el;
         // seu código vem aqui
      }

      plugin.fooPublicMethod = function() {
         // seu código vem aqui
      }

      var fooPrivateMethod = function() {
         // seu código vem aqui
      }

      init();

   }

})(jQuery);
